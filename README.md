# GPA Calculator

GPA Calculator is a simple and user-friendly web application that allows students to calculate their Grade Point Average (GPA) quickly and accurately. Built using **HTML, CSS, and JavaScript**, this tool provides an easy interface to enter subjects, grades, and credit hours, and instantly get the GPA. The project is designed to be lightweight, responsive, and accessible for students of all levels.
